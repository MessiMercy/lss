

# 1、MySQL数据库版本5.7需要驱动8以上

```xml
  <!-- mysql 5.7的数据库需要8的驱动 -->
        <dependency>
             <groupId>mysql</groupId>
             <artifactId>mysql-connector-java</artifactId>
             <version>8.0.11</version>
         </dependency>
```

url配置：

useSSL=false&characterEncoding=utf-8&serverTimezone=UTC
